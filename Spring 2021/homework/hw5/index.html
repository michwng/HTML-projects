Overview

HTML (HyperText Markup Language) is a programming language developed by Tim Berners-Lee in 1990. It is used to create webpages and is often paired with CSS and Javascript to create remarkable webpages. HTML is regulated by the World Wide Web Consortium (W3C) and the most recent standard of HTML is HTML5.

Here are a few tags used in HTML.

  • <html> and </html>
  • <head> and </head>
  • <body> and </body>
  • <p> and </p>
  • <article> and </article>

HTML Tags, most of the time, consist of an opening and closing tag. In the case of <html> and </html>, <html> is the opening tag and </html> is the closing tag. However, if you put this, some elements, and text without CSS, it will look bland on the webpage. With CSS, you can make the website even more visually appealing!
