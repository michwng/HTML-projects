Block-Level Elements

Block-Level elements are the big boys of the code. In other words, there can be in-line and block elements inside a block-level element. Yes, you heard me right. There can be a Block-Level Element inside another Block-Level Element.

Here are a few Block-Level tags used in HTML.

Term Opening Tag Closing Tag What It Does
html <html> </html> Houses code for the HTML document.
head <head> </head> Stores metadata about the webpage.
body <body> </body> What you see on the website. (Content)
p <p> </p> Displays text.
div <div> </div> Acts as a section of the content.

Again, these tags are big boi tags. Thus, they are called block-level elements. You won't be able to find a tag with a higher level than this. They can hold other block-level elements inside them, as well as in-line elements.
