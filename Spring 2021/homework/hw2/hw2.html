Welcome to HTML

HTML is an acronym for Hypertext Markup Language. It is also known as "The Language of the Web." HTML is the brainchild of Sir Time Berners-Lee, who also developed the Hypertext Transfer Protocoland Uniform Resource Identifiers. Together the three technologies make the Web possible.

HTML is a plain-text language. This means that web pages are human readable. The "Markup" part of HTML simply means that we use characters that have a special meaning for web browsers to modify how the browser presents the content. These special characters are called "tags." Tags, coupled with their content, are referred to as "elements."

The paragraph element is one of the basic building blocks of HTML. It is a block level element, which means that it breaks the flow of the content of the page as it is rendered and adds whitespace before and after.

Basic HTML

There are a number of HTML tags and elements that we will explore this semester, including anchor tags (for hyperlinks), img (for images), strong, em, b, i, ul (lists), ol (numbered lists), hr (horizontal rule), br (link break), divs, spans, and many, many more.
